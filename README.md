wefunk_dl
===============

Download a show from the WeFunk website: http://www.wefunkradio.com/
WeFunk is the best Hip-Hop, Soul and Funk radio on earth.

Prerequisites
-------------

* Python 3

Usage examples
--------------

This will download the show 800:

    ./wefunk_dl 800
